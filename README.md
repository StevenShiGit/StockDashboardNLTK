# Stock Dashboard NLTK

This is a stock dashboard full-stack application that displays market analytics that displays stock information as well as overall stock sentiment based on recent news articles about the specific stock.
It also displays the recent news articles with links and sentiment so the user can read the articles for themselves.

It obtains the stock data through third-party REST API's with HTTP methods. The frotend is built with Javascript and REACT and NodeJS, and it's styled with Tailwind CSS.

The backend for sentiment analysis uses NLP and is built with Python's Natural Language Toolkit. It also uses Beutiful Soup and Pandas for webscraping, and NumPy for data analysis
I built a REST API with Flask that returns JSON data containing news sentiment, links to articles, and headlines about given stock.

## Technologies Used
- Node.JS
- Javascript
- React
- HTML/CSS, Tailwind
- Python
- Flask
- Python NLTK, NumPy, Pandas, Beutiful Soup
- REST API
